## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/nft-site.git
```

Go to the project directory

```bash
  cd ..nft-site
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run start
```
